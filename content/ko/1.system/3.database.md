# 최종 데이터베이스 설계 명세서

## 개요
이 애플리케이션은 Drizzle ORM을 통해 관리되는 PostgreSQL 데이터베이스를 사용합니다.

## 스키마 (Schema)
### Users 테이블
- `id`: UUID (기본 키)
- `email`: VARCHAR (고유 값)
- `name`: VARCHAR

### Properties 테이블
- `id`: UUID
- ...
