# 시스템 구성 문서

## 환경 (Environment)
- **Node.js**: v18+
- **Nuxt**: v4.x
- **Database**: PostgreSQL (Supabase/로컬 환경)

## 환경 변수 (Environment Variables)
- `DATABASE_URL`: PostgreSQL 연결 문자열
- `NUXT_PUBLIC_API_BASE`: API 기본 URL

(하드웨어 사양 및 소프트웨어 버전 정보는 여기에 작성됩니다)
