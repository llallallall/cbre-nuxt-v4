# 최종 소스 코드

## 개요
본 문서는 CBRE Nuxt v4 프로젝트의 최종 인도 소스 코드 구조를 설명합니다.

## 리포지토리 구조
- `app/`: 메인 애플리케이션 소스 코드 (Nuxt 4 구조)
- `server/`: API 및 서버 사이드 로직
- `content/`: 문서 콘텐츠 (현재 디렉토리)

## 주요 컴포넌트
- **TopBar**: `app/components/nav/TopBar.vue`
- **Profile**: `app/components/nav/user/Profile.vue`
- ...

(개발 코드 및 세부 사항은 여기에 작성됩니다)
