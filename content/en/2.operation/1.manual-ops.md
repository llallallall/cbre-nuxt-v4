# Operation & Maintenance Manual

## Server Management
- **Start**: `npm run start`
- **Restart**: `pm2 restart cbre-nuxt`

## Backup & Recovery
- **Database**: Daily backups scheduled via cron.
- **Recovery**: Restore from S3 bucket `cbre-backups`.

## Maintenance
- Check logs at `/var/log/cbre/app.log`
