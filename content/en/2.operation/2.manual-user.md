# User Manual

## Getting Started
To access the system, navigate to the homepage and log in with your credentials.

## Features
### Profile Management
1. Click the user icon in the top right.
2. Select **Edit Profile**.
3. Update your details and click **Save**.

### Map Visualization
- Use the toggle filters to change view modes.
- Click on markers to see property details.
