# Cross-Agent/PC Workflow Guide

## Overview
Guidelines for maintaining consistency when switching between PCs or Agents.

## Workflow
1. **Check Instructions**: Always check `instructions/` folder before starting work.
2. **Check Task History**: Review `task.md` and `INSTRUCTION_MIGRATION_HISTORY.md`.
3. **Sync State**: Ensure you have the latest code (git pull) and dependencies (npm install).
4. **Consistency**:
   - Follow `INSTRUCTION_UI_STANDARD.md` for UI changes.
   - Follow `INSTRUCTION_AUTH.md` for auth changes.
5. **Documentation**:
   - Update `INSTRUCTION_MIGRATION_HISTORY.md` if you make significant architectural changes.
   - Update `task.md` with progress.
